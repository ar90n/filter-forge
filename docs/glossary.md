# FilterForge 用語集

## 1. フィルタ基礎

| 日本語 | English | 定義 |
|--------|---------|------|
| フィルタ | Filter | 信号の特定の周波数成分を選択的に通過または減衰させる回路またはシステム |
| パッシブフィルタ | Passive Filter | 受動素子（抵抗 R、インダクタ L、コンデンサ C）のみで構成されるフィルタ回路。外部電源を必要としない |
| アクティブフィルタ | Active Filter | オペアンプ等の能動素子を含むフィルタ回路。外部電源が必要。MVP スコープ外 |
| ローパスフィルタ (LPF) | Low-Pass Filter | カットオフ周波数以下の低周波数成分を通過させ、それより高い周波数を減衰させるフィルタ |
| ハイパスフィルタ (HPF) | High-Pass Filter | カットオフ周波数以上の高周波数成分を通過させ、それより低い周波数を減衰させるフィルタ |
| バンドパスフィルタ (BPF) | Band-Pass Filter | 特定の周波数帯域のみを通過させ、それ以外の周波数を減衰させるフィルタ |
| バンドエリミネートフィルタ (BEF) | Band-Elimination Filter | 特定の周波数帯域を除去（減衰）し、それ以外の周波数を通過させるフィルタ。ノッチフィルタとも呼ばれる |
| オールパスフィルタ (APF) | All-Pass Filter | 全周波数帯域で振幅特性を変えず、位相のみを変化させるフィルタ。群遅延の補正や位相調整に使用 |

## 2. フィルタ特性

| 日本語 | English | 定義 |
|--------|---------|------|
| カットオフ周波数 | Cutoff Frequency | フィルタの通過帯域と遮断帯域の境界となる周波数。通常 -3dB 点として定義される |
| 中心周波数 | Center Frequency | BPF/BEF における通過帯域（または阻止帯域）の幾何平均中心の周波数 |
| 帯域幅 | Bandwidth | BPF/BEF における -3dB 点間の周波数幅 |
| フィルタ次数 | Filter Order | フィルタの伝達関数の次数。次数が高いほど遮断特性が急峻になる |
| 通過帯域 | Passband | フィルタが信号を通過させる周波数範囲 |
| 阻止帯域 | Stopband | フィルタが信号を減衰させる周波数範囲 |
| 遷移帯域 | Transition Band | 通過帯域と阻止帯域の間の周波数範囲 |
| リップル | Ripple | 通過帯域または阻止帯域における振幅特性の波打ち。dB 単位で表す |
| 通過帯域リップル | Passband Ripple | 通過帯域内での振幅の最大偏差。Chebyshev Type I および Elliptic フィルタで生じる |
| 阻止帯域減衰量 | Stopband Attenuation | 阻止帯域における最小減衰量。dB 単位。Chebyshev Type II および Elliptic フィルタで指定 |
| 群遅延 | Group Delay | 信号の包絡線が回路を通過するのにかかる時間。位相特性の周波数微分の負値 |
| 挿入損失 | Insertion Loss | フィルタを回路に挿入したことによる信号電力の損失。dB 単位 |

## 3. 近似関数

| 日本語 | English | 定義 |
|--------|---------|------|
| バターワース近似 | Butterworth Approximation | 通過帯域で最大限平坦な振幅特性を持つフィルタ近似。リップルなし。遷移帯域はやや緩やか |
| チェビシェフ Type I 近似 | Chebyshev Type I Approximation | 通過帯域に等リップルを持つフィルタ近似。同じ次数のバターワースより急峻な遮断特性を持つ |
| チェビシェフ Type II 近似 | Chebyshev Type II Approximation | 阻止帯域に等リップルを持つフィルタ近似。通過帯域は平坦。逆チェビシェフとも呼ばれる |
| ベッセル近似 | Bessel Approximation | 通過帯域で最大限平坦な群遅延特性を持つフィルタ近似。位相直線性が最も良い。遮断特性は緩やか |
| 楕円（カウアー）近似 | Elliptic (Cauer) Approximation | 通過帯域と阻止帯域の両方に等リップルを持つフィルタ近似。同じ次数で最も急峻な遮断特性を実現 |

## 4. 回路・部品

| 日本語 | English | 定義 |
|--------|---------|------|
| ラダー型回路 | Ladder Network | 直列素子と並列素子を交互に配置した回路構成。パッシブフィルタの基本的な実現形式 |
| T 型回路 | T-Network | ラダー型の一種。直列素子 → 並列素子 → 直列素子の順で始まる構成 |
| π 型回路 | Pi-Network | ラダー型の一種。並列素子 → 直列素子 → 並列素子の順で始まる構成 |
| ラティス型回路 | Lattice Network | ブリッジ構成で実現する回路。主にオールパスフィルタの実現に使用 |
| コンデンサ | Capacitor | 電荷を蓄える受動素子。単位はファラド (F)。高周波成分を通過させる |
| インダクタ | Inductor | 磁気エネルギーを蓄える受動素子。単位はヘンリー (H)。低周波成分を通過させる |
| 抵抗 | Resistor | 電流の流れを制限する受動素子。単位はオーム (Ω) |
| ソースインピーダンス | Source Impedance | 信号源の出力インピーダンス。フィルタの入力側に接続される |
| 負荷インピーダンス | Load Impedance | 負荷の入力インピーダンス。フィルタの出力側に接続される |
| BOM | Bill of Materials | 部品表。製品を構成する部品の一覧。部品記号、種別、値、数量を含む |

## 5. 信号処理・数学

| 日本語 | English | 定義 |
|--------|---------|------|
| 伝達関数 | Transfer Function | システムの入出力関係を周波数領域で表す数学的関数。H(s) = B(s)/A(s) の形式 |
| 周波数応答 | Frequency Response | フィルタの入出力特性を周波数の関数として表したもの。振幅特性と位相特性からなる |
| ボード線図 | Bode Plot | 周波数応答を対数周波数軸上にゲイン (dB) と位相 (度) で表示するグラフ |
| 振幅特性 | Magnitude Response | 周波数の関数としてのフィルタのゲイン（利得）。通常 dB で表示 |
| 位相特性 | Phase Response | 周波数の関数としてのフィルタの位相シフト。通常度 (°) で表示 |
| インパルス応答 | Impulse Response | インパルス信号（デルタ関数）を入力したときのシステムの出力応答 |
| ステップ応答 | Step Response | ステップ信号を入力したときのシステムの出力応答 |
| プロトタイプフィルタ | Prototype Filter | カットオフ周波数 1 rad/s、インピーダンス 1 Ω に正規化されたフィルタ。設計の基準となる |
| 周波数変換 | Frequency Transformation | プロトタイプフィルタ（LPF）から他の種別（HPF/BPF/BEF/APF）へ変換する数学的操作 |
| インピーダンススケーリング | Impedance Scaling | プロトタイプフィルタの素子値を実際のインピーダンスレベルに変換する操作 |
| ラプラス変換 | Laplace Transform | 時間領域の関数を複素周波数領域（s 領域）に変換する積分変換。伝達関数の基礎 |
| デシベル (dB) | Decibel | 信号の電力比または振幅比を対数スケールで表す単位。20·log10(V_out/V_in) |
